# console-node-javascript

Ejecutar

Descarga paquetería necesaria para ejecución
npm i

Instala de manera global
npm install -g .

Ejecuta app
hello -n Panchito ó hello --name Panchito

Muestra ayuda
hello --help

Muestra versión
hello --version

Desinstalar
npm uninstall -g console-node-javascript
